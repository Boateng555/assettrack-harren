Sync Instructions

How to Set Up Azure AD Integration:

  1. Register an application in Azure AD
  2. Grant appropriate permissions (User.Read.All, Device.Read.All)
  3. Add the following settings to your Django settings.py:
# Azure AD Integration Settings
AZURE_TENANT_ID = 'your-tenant-id'
AZURE_CLIENT_ID = 'your-client-id'
AZURE_CLIENT_SECRET = 'your-client-secret'

Available Commands:

# Full sync
python manage.py sync_azure_ad

# Sync only employees
python manage.py sync_azure_ad --employees-only

# Sync only devices
python manage.py sync_azure_ad --devices-only

# Sync only assignments
python manage.py sync_azure_ad --assignments-only
